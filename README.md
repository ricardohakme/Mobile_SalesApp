# Mobile Sales App (Delphi FMX)

Aplicativo de vendas externas desenvolvido em Delphi FMX, usando arquitetura MVVM simplificada.

## Funcionalidades
- Listagem de clientes com filtro em tempo real
- Criação e envio de pedidos simulados
- Layout profissional com ToolBar, ListBox e botões estilizados
- Estrutura modular preparada para integração REST ou banco de dados

## Tecnologias
- Delphi FMX
- Arquitetura MVVM
- Layout responsivo e organizado
- Mock de dados corporativos recriados para portfólio

## Objetivo
Demonstrar habilidades em desenvolvimento mobile corporativo, integração de sistemas e organização de código profissional.

## Observação
Projeto baseado em experiências reais, com código e dados recriados para fins de portfólio.
